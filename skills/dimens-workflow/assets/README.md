# dimens-workflow assets

当前目录用于存放 `dimens-workflow` 在 ClawHub / OpenClaw 发布时使用的视觉素材。

## 推荐文件名

- `cover.png`
- `icon.png`
- `screenshot-01.png`
- `screenshot-02.png`

## 当前最小要求

当前没有素材时，可以只保留本 `README.md`。

如果准备补最小可发布素材，建议至少补：

- `cover.png`
- `icon.png`

## 素材建议内容

- `cover.png`：突出工作流、审批、自动化、AI 分析
- `icon.png`：突出流程节点、自动化链路
- `screenshot-01.png`：展示工作流调用或挂载场景
- `screenshot-02.png`：展示模型路由或项目绑定场景

通用命名、尺寸和格式规范统一参考：
