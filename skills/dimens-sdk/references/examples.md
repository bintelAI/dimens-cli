## 推荐阅读顺序

如果你是第一次进入 `dimens-sdk`，优先按下面顺序看：

1. 前端接入总览：`references/frontend/overview.md`
2. 前端首次接入：`references/frontend-quickstart.md`
3. 登录与 token 主链：`references/frontend-auth-flow.md`
4. React / Vue 落地示例：`references/react-auth-example.md`
5. 最佳实践：`references/sdk-best-practices.md`
6. refresh / retry：`references/request-retry-example.md`

如果你不是前端接入，而是按资源域排查或接入，直接进入对应文档：

- 表格：`references/table-examples.md`
- 文档：`references/document-examples.md`
- 报表：`references/report-examples.md`
- 上传：`references/upload-examples.md`
- AI：`references/ai-examples.md`
- Web/H5：`references/web-examples.md`
- 移动端：`references/mobile-examples.md`
- BFF：`references/bff-examples.md`

如果你要开发或排查 `dimens-web` 微模块，按下面顺序看：

1. 脚手架基础：`references/dimens-web-scaffold.md`
2. 三类微模块上下文协议：`references/micro-module-wujie-context.md`
3. 需要读取表格数据时再看：`references/table-examples.md`
4. 涉及 BFF 或密钥托管时再看：`references/bff/overview.md`
