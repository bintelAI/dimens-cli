# dimens-key-auth assets

当前目录用于存放 `dimens-key-auth` 在 ClawHub / OpenClaw 发布时使用的视觉素材。

## 推荐文件名

- `cover.png`
- `icon.png`
- `screenshot-01.png`
- `screenshot-02.png`

## 当前最小要求

当前没有素材时，可以只保留本 `README.md`。

如果准备补最小可发布素材，建议至少补：

- `cover.png`
- `icon.png`

## 素材建议内容

- `cover.png`：突出 API Key、token、外部接入
- `icon.png`：突出登录、密钥、鉴权语义
- `screenshot-01.png`：展示 Key 登录或管理场景
- `screenshot-02.png`：展示第三方接入或 token 复用场景
