# dimens-system-orchestrator assets

当前目录用于存放 `dimens-system-orchestrator` 在 ClawHub / OpenClaw 发布时使用的视觉素材。

## 推荐文件名

- `cover.png`
- `icon.png`
- `screenshot-01.png`
- `screenshot-02.png`
- `screenshot-03.png`

## 当前最小要求

当前没有素材时，可以只保留本 `README.md`。

如果准备补最小可发布素材，建议至少补：

- `cover.png`
- `icon.png`

## 素材建议内容

- `cover.png`：突出系统搭建、模块拆解、技能路由
- `icon.png`：突出总控、编排、中枢入口
- `screenshot-01.png`：展示系统拆解输出骨架
- `screenshot-02.png`：展示技能路由或命令映射
- `screenshot-03.png`：展示从项目到表到权限的整条主链
