# dimens-project assets

当前目录用于存放 `dimens-project` 在 ClawHub / OpenClaw 发布时使用的视觉素材。

## 推荐文件名

- `cover.png`
- `icon.png`
- `screenshot-01.png`
- `screenshot-02.png`

## 当前最小要求

当前没有素材时，可以只保留本 `README.md`。

如果准备补最小可发布素材，建议至少补：

- `cover.png`
- `icon.png`

## 素材建议内容

- `cover.png`：突出项目创建、初始化、默认公开视图
- `icon.png`：突出项目容器、启动入口
- `screenshot-01.png`：展示项目创建或初始化主链
- `screenshot-02.png`：展示默认公开视图补齐或项目到建表的推进链路

通用命名、尺寸和格式规范统一参考：
