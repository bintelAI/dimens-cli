# dimens-table assets

当前目录用于存放 `dimens-table` 在 ClawHub / OpenClaw 发布时使用的视觉素材。

## 推荐文件名

- `cover.png`
- `icon.png`
- `screenshot-01.png`
- `screenshot-02.png`
- `screenshot-03.png`

## 当前最小要求

当前没有素材时，可以只保留本 `README.md`。

如果准备补最小可发布素材，建议至少补：

- `cover.png`
- `icon.png`

## 素材建议内容

- `cover.png`：突出表、字段、视图、行数据
- `icon.png`：突出表格、字段列、数据结构
- `screenshot-01.png`：展示建表或字段设计
- `screenshot-02.png`：展示默认视图、筛选或 `row/page`
- `screenshot-03.png`：展示 relation、select 选项或典型数据链路
