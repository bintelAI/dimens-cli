# dimens-report assets

当前目录用于存放 `dimens-report` 在 ClawHub / OpenClaw 发布时使用的视觉素材。

## 推荐文件名

- `cover.png`
- `icon.png`
- `screenshot-01.png`
- `screenshot-02.png`

## 当前最小要求

当前没有素材时，可以只保留本 `README.md`。

如果准备补最小可发布素材，建议至少补：

- `cover.png`
- `icon.png`

## 素材建议内容

- `cover.png`：突出报表、图表、参数联动
- `icon.png`：突出图表、分析看板、数据查询
- `screenshot-01.png`：展示报表或图表场景
- `screenshot-02.png`：展示参数联动或数据源查询场景
