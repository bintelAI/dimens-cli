# dimens-team assets

当前目录用于存放 `dimens-team` 在 ClawHub / OpenClaw 发布时使用的视觉素材。

## 推荐文件名

- `cover.png`
- `icon.png`
- `screenshot-01.png`
- `screenshot-02.png`

## 当前最小要求

当前没有素材时，可以只保留本 `README.md`。

如果准备补最小可发布素材，建议至少补：

- `cover.png`
- `icon.png`

## 素材建议内容

- `cover.png`：突出团队、项目上下文、隔离边界
- `icon.png`：突出空间、成员、上下文入口
- `screenshot-01.png`：展示团队与项目上下文判断
- `screenshot-02.png`：展示项目入口或上下文来源示例
