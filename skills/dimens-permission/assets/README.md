# dimens-permission assets

当前目录用于存放 `dimens-permission` 在 ClawHub / OpenClaw 发布时使用的视觉素材。

## 推荐文件名

- `cover.png`
- `icon.png`
- `screenshot-01.png`
- `screenshot-02.png`

## 当前最小要求

当前没有素材时，可以只保留本 `README.md`。

如果准备补最小可发布素材，建议至少补：

- `cover.png`
- `icon.png`

## 素材建议内容

- `cover.png`：突出角色、项目权限、访问控制
- `icon.png`：突出权限矩阵、锁、分层控制
- `screenshot-01.png`：展示角色创建或权限主链
- `screenshot-02.png`：展示项目权限、资源权限或行级策略
